# Mister Coffee Landing Page Premium

Landing page premium, responsive y animada para **Mister Coffee**, cafetería artesanal ubicada en La Troncal, Cañar, Ecuador.

El sitio ha sido diseñado con una estética limpia, cálida y de alta gama (estilo Awwwards/Framer), optimizando el copywriting y los flujos de conversión local para dispositivos móviles y de escritorio.

---

## 🚀 Tecnologías y Stack Principal
- **Core**: Next.js, React, TypeScript, Tailwind CSS, Framer Motion, GSAP, Lenis.

---

## ✨ Características Principales
- **Diseño premium glassmorphism**: Composición asimétrica, amplio espaciado, variables HSL cálidas y grano orgánico táctil.
- **Animaciones suaves**: Revelado de scroll, parallax y transiciones cinemáticas fluidas.
- **Menú interactivo**: Filtro por categorías dinámico, buscador integrado en tiempo real y enlaces de pedido individualizados por plato.
- **Galería responsive**: Grilla modular masonry adaptativa para cualquier tamaño de pantalla.
- **Video promocional vertical**: Loop de video 9:16 en un elegante smartphone mockup.
- **Mapa interactivo**: Google Maps integrado y estilizado con filtros CSS modernos.
- **CTA a WhatsApp**: Botón flotante widget y botones principales con redirecciones directas.
- **SEO local**: Estructura semántica HTML5, metadatos enriquecidos y esquema estructurado JSON-LD (`CafeOrCoffeeShop`).
- **Responsive mobile-first**: Experiencia de usuario impecable optimizada para pantallas táctiles desde 360px hasta resoluciones de escritorio.

## 🛠️ Comandos de Ejecución y Despliegue

### 1. Instalación de Dependencias
```bash
npm install
```

### 2. Ejecutar Servidor Local de Desarrollo
```bash
npm run dev
```
*Abre [http://localhost:3000](http://localhost:3000) en tu navegador.*

### 3. Compilación de Producción
```bash
npm run build
```

## 🛡️ Despliegue en Vercel
Este proyecto está optimizado y listo para ser desplegado en Vercel:
1. Sube el proyecto a tu repositorio de GitHub.
2. Inicia sesión en [Vercel](https://vercel.com/) e importa tu repositorio `cafeteriapremiun`.
3. Configura los parámetros por defecto:
   - **Framework Preset**: Next.js
   - **Build Command**: `npm run build`
   - **Output Directory**: Next.js default (`.next`)
4. Haz clic en **Deploy**. ¡Listo!
